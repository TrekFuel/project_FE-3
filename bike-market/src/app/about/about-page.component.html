Данная торговая площадка является Single Page Application, написанным при помощи фреймворка Angular. При этом содержит следующий функционал:

  • главная страница товаров (с превью всех товаров);
  • страница товара по id (подробная страница отдельного товара);
  • фильтрация товаров по списку параметров (чекбоксы);
  • страница "О проекте";
  • функционал добавления объявления на площадке;
  • функционал входа;
  • функционал регистрации.
